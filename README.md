# Тестовое задание для стажёра Frontend (весенняя волна 2026)

## Как запустить проект

1. npm install
2. npm run dev

## Как запустить сервер
1. cd server
2. npm i
3. npm start
После этого сервер будет запущен на порту 8080. Если нужно запустить его на другом порту, это можно сделать при помощи переменной окружения PORT - PORT=8080 npm start.

## Как запустить serverllm

Для запуска `serverllm`, выполните следующие шаги:

1. Убедитесь, что у вас установлен Node.js и npm.
2. Перейдите в директорию `serverllm`:
   ```bash
   cd serverllm
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер:
   ```bash
   npm start
   ```

### Необходимые ключи

Для работы `serverllm` может потребоваться API ключ от Гигачата. Убедитесь, что вы добавили его в файл конфигурации или передали через переменные окружения. Обратитесь к документации Гигачата для получения информации о том, как получить API ключ.
